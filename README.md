# Piano Repertoire Explorer

What people actually played: 264,096 piano works published 1820-1910, reconstructed from national-library and music-trade catalogues across nine markets - an alternative, evidence-based history of the domestic piano repertoire, set against the tiny canon-formation layer (635 anthology/revival selections) that decided what "deserved" to survive.

Live site: the GitHub Pages deployment of index.html. Start with the Top 20 by Year tab and press Play. The Dissemination Map tab traces documented city-by-city spreads.

Companion project: Women at the Keys (https://josebowen.github.io/women-at-the-keys/) - 477 women composers of piano music.

Core caveat, disclosed throughout: publication does not equal sales does not equal playing. Counts are circulation evidence with archive-depth bias (the German trade is over-documented relative to other markets). Full source criticism on the site's Sources & Methods page.

corpus_digest_v41.json is a machine-readable summary (totals, distributions, top pieces, method caveats) used by the site's "Ask Claude" button.
